------------------------------------------------------Commit na master--------------------------------------------------------

git init
git add --all
git commit -m "Initial Commit"
git remote add origin ORIGIN_URL 
git push -u origin master

Where ORIGIN_URL is your repository url (e.g. https://username@your.bitbucket.domain:7999 /yourproject/repo.git) [Obs: clonar o repositório]

-----------------------------------------------------Como fazer um push numa branch existente--------------------------------------------------------------------------------------------
Vá até a pasta, que deve estar como remote configurado (origin)

git add . - para adicionar tudo que foi alterado ou criado (menos os deletados)
git status - para confirmar as ações do git add
git commit -m "Features da v2" - para inserir as alterações
git push origin test
